## 字体特点 Font speciality
荆南波波黑横竖略带波浪，不完全平直，尽显俏皮可爱。
![explain01](./documentation/explain01.jpg)

口左上角的出头视情况而定。  
精心设计的日文假名和注音符号。  
流畅的曲线彰显个性动感。

![explain02](./documentation/explain02.png)

字体笔画粗细排布均一合理，小字可读性较高。
